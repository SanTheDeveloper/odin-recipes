# Odin Recipes

This is my first project from The Odin Project's Foundations curriculum. In this project, I am building a basic recipe website to practice and demonstrate my foundational web development skills.

## Project Goals

The goal of this project is to build a multi-page website using raw HTML and to practice version control using Git and GitHub.

## Skills Demonstrated

- Setting up a standard HTML boilerplate.
- Structuring text using paragraphs and hierarchical headings.
- Utilizing ordered (`<ol>`) and unordered (`<ul>`) lists for ingredients and instructions.
- Embedding images and writing proper `alt` text.
- Using anchor tags (`<a>`) with relative links to navigate between multiple HTML pages within a directory.
- Using the command line to stage and commit changes with descriptive commit messages.
